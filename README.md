zaw-src-git-log
===============

This script is zaw source to get commit hash from git log.

Installation
------------

Download the source from http://github.com/yonchu/zaw-src-git-log
and source it in your ``.zshrc``.

```sh
source zaw-src-git-log/zaw-git-log.zsh
```
